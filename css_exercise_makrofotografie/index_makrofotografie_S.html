Makrofotografie - Definition


Als Makrofotografie (auch „Nahfotografie“) wird das Fotografieren von Objekten mit einem ungefähren Maßstab von 1:1 bezeichnet. Das bedeutet dass sich das Motiv mehr oder weniger in Originalgröße auf dem Kamerasensor festgehalten wird. Vorraussetzung für erfolgreiche Makrofotografie ist (neben der richtigen Ausrüstung) eine geringe Entfernung zwischen Kamera und Motiv. Nur so gelingen beeindruckende, detailstarke Aufnahmen.

Technik der Makrofotografie


Der Begriff ‚Makro‘ wird mittlerweile salopp auch für viele Objektive gebraucht, die eine vergleichsweise geringe Minimale Objektdistanz MOD zulassen, so auch bspw. für das Tamron Superzoom 18-270. Tatsächlich ist die Bedeutung dieses Begriffes aber festgelegt auf einen Abbildungsmaßstab größergleich 1:1. Im Detail heißt dies, dass das Objekt in einer bestimmten Größenausdehnung (Breite x Höhe) im Maßstab [mm] auch mindestens in dieser Größe (Breite x Höhe in [mm]) auf dem Film oder dem Sensor der Kamera erscheint. Je nach Größe des Sensors (4/3, APS-C, Kleinbild, Mittelformat …) ist das Objekt damit folglich auch mehr oder weniger bildfüllend.
